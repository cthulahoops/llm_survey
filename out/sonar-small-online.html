I need to take this json:

{
'first_name': 'Jane',
'last_name': 'Smith',
'date_of_birth': '03/16/1977',
orders: [
{
'created': '2024-01-25T15:00:04',
'amount': '100.00'
}
]}

And generate some dataclasses.

date_of_birth needs to be a date.
created is a datetime.
amount is a decimal.

How would I do this using the marshmallow library in python?

Response 1.

You can use the marshmallow-dataclass library to automatically generate marshmallow schemas from your dataclasses. First, install the library using pip:

pip install marshmallow-dataclass

Then, import the dataclass module, define your dataclass, and use the marshmallow schema. Here is an example:

from marshmallow_dataclass import dataclass, fields
from dataclasses import dataclass
import datetime

@dataclass
class Order:
    created: fields.DateTime()
    amount: fields.Decimal()

@dataclass
class Person:
    first_name: str
    last_name: str
    date_of_birth: datetime.date
    orders: list[Order]

person = Person("Jane", "Smith", "1977-03-16", [Order("2024-01-25T15:00:04", "100.00")])
schema = Person.Schema()
data = schema.dump(person)
print(data)  # Output: {'first_name': 'Jane', ...

In this example, the Order class represents an order with a created datetime and amount decimal field. The Person class represents a person with a date_of_birth date field and a list of orders. The person instance is then serialized using the schema, which generates a dictionary containing the values of the fields specified in the dataclass.
