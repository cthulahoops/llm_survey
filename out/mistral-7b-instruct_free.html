from pydantic import BaseModel, Date, DateTime, Decimal
from typing import List

class Order(BaseModel):
    created: DateTime
    amount: Decimal

class Person(BaseModel):
    first_name: str
    last_name: str
    date_of_birth: Date
    orders: List[Order]

person = Person(**data)

print(person.first_name) # Output: Jane
print(person.last_name) # Output: Smith
print(person.date_of_birth) # Output: 1977-03-16
print(person.orders) # Output: <Order(created=datetime.datetime(2024, 1, 25, 15, 0, tzinfo=<TimeZone 'utc'>), amount=100.0)>

The Order class is used to form the orders list, and the Person class is used to form the rest of the data. The BaseModel class provides a base class for the dataclass, and it has attributes such as first_name, last_name, etc.

The Date and DateTime classes from the pydantic module are used to convert the date_of_birth and created attributes to date and datetime objects, respectively.

The Decimal class from the pydantic module is used to convert the amount attribute to a decimal object. The typing.List class is used for the orders list in order to include the Order dataclass objects.
