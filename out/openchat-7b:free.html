To create dataclasses using marshmallow, you'll first need to install the dataclasses-marshmallow package by running:

pip install dataclasses-marshmallow

Here's an example of how you can define your dataclasses using marshmallow and the dataclasses-marshmallow package:

from dataclasses import dataclass
from marshmallow import Schema, fields, post_load
from marshmallow_dataclass import dataclass
from datetime import datetime
from dateutil.relativedelta import relativedelta
from decimal import Decimal

@dataclass
class Order:
    created: datetime
    amount: Decimal

@dataclass
class Customer:
    first_name: str
    last_name: str
    date_of_birth: datetime
    orders: List[Order]

class CustomerSchema(Schema):
    first_name = fields.Str()
    last_name = fields.Str()
    date_of_birth = fields.Str(converter=lambda x: datetime.strptime(x, "%m/%d/%Y").date())
    orders = fields.Nested(OrderSchema, many=True)

    @post_load
    def make_customer(self, data, **kwargs):
        return Customer(**data)

class OrderSchema(Schema):
    created = fields.DateTime(format="%Y-%m-%dT%H:%M:%S")
    amount = fields.Decimal(as_string=True)

def main():
    json_data = {
        "first_name": "Jane",
        "last_name": "Smith",
        "date_of_birth": "03/16/1977",
        "orders": [
            {
                "created": "2024-01-25T15:00:04",
                "amount": "100.00"
            }
        ]
    }

    customer_schema = CustomerSchema()
    customer = customer_schema.load(json_data)

    order_schema = OrderSchema()
    orders = [order_schema.load(order_data) for order_data in customer.orders]

    print(customer)

if __name__ == "__main__":
    main()

In this example, we define two dataclasses: Customer and Order. The CustomerSchema and OrderSchema classes define how to serialize and deserialize these dataclasses using marshmallow. We use custom converters to parse the date_of_birth and created fields from strings to the appropriate date and datetime types, respectively.

The make_customer method in the CustomerSchema class takes the deserialized data and creates a Customer instance using the **data syntax.

Finally, we use the CustomerSchema and OrderSchema to load the JSON data and print the resulting Customer instance.
