Prompt

I need to take this json:

{
'first_name': 'Jane',
'last_name': 'Smith',
'date_of_birth': '03/16/1977',
orders: [
{
'created': '2024-01-25T15:00:04',
'amount': '100.00'
}
]}

And generate some dataclasses.

date_of_birth needs to be a date.
created is a datetime.
amount is a decimal.

How would I do this using the marshmallow library in python?

Response 1.

Sure, here is how you would do it using Marshmallow. First of all, you would need to import the necessary modules and classes:

from dataclasses import dataclass
from typing import List
from datetime import date, datetime
from decimal import Decimal
from marshmallow import fields, post_load
from marshmallow_dataclass import class_schema

Then, you would define your data classes:

@dataclass
class Order:
    created: datetime
    amount: Decimal


@dataclass
class Person:
    first_name: str
    last_name: str
    date_of_birth: date
    orders: List[Order]

In these class definitions, created, amount, and date_of_birth are typed as datetime,Decimal, and date as per your specifications.

Now, you want to deserialize JSON into these dataclasses. Marshmallow dataclass can generate a schema automatically from the dataclasses:

PersonSchema = class_schema(Person)()

class_schema generates a schema which we can use to load/validate/dump data. It also handles nested fields, so the schema for 'Order' is automatically generated and included in the 'Person' schema.

Since created, amount, and date_of_birth require custom parsing, you'll need to override these fields using a marshmallow schema:

class CustomPersonSchema(PersonSchema):
    date_of_birth = fields.Date()
    orders = fields.Nested(class_schema(Order)(), many=True)


class CustomOrderSchema(class_schema(Order)()):
    created = fields.DateTime()
    amount = fields.Decimal(as_string=True)

Finally, this is how you deserialize a person from JSON:

json_data = {
    'first_name': 'Jane',
    'last_name': 'Smith',
    'date_of_birth': '1977-03-16',
    'orders': [
        {
            'created': '2024-01-25T15:00:04',
            'amount': '100.00'
        }
    ]
}

person, errors = CustomPersonSchema().load(json_data)

In this case person is now an instance of Person, with date_of_birth correctly parsed as a date, and each order in orders is an instance of Order, with created as a datetime and amount as decimal.

Note: the as_string=True argument for the Decimal field. If False (the default), the field will deserialize to decimal.Decimal objects. If True, the field will deserialize to strings.

import marshmallow_dataclass
from marshmallow import Schema

# The function automatically creates a marshmallow schema for your dataclass.
OrderSchema = marshmallow_dataclass.class_schema(Order)
UserSchema = marshmallow_dataclass.class_schema(User)

# Here is an example of how to use these schemas for serialization/deserialization.

user_data = {
    'first_name': 'Jane',
    'last_name': 'Smith',
    'date_of_birth': '03/16/1977',
    'orders': [
        {
            'created': '2024-01-25T15:00:04',

        }

Response 3.

Firstly, let's import necessary modules including datetime, dataclasses, and marshmallow. Then, create dataclasses representing your data. Lastly, you can use marshmallow_dataclass to automatically generate Marshmallow schemas from these dataclasses.

Here's what the code would look like:

from decimal import Decimal
from dataclasses import dataclass
from datetime import datetime, date
from typing import List
from marshmallow import Schema, fields, post_load

@dataclass
class Order:
    created: datetime
    amount: Decimal

@dataclass
class Person:
    first_name: str
    last_name: str
    date_of_birth: date
    orders: List[Order]

class OrderSchema(Schema):
    created = fields.DateTime(required=True)
    amount = fields.Decimal(as_string=True)

    @post_load
    def make_order(self, data, **kwargs):
        return Order(**data)

class PersonSchema(Schema):
    first_name = fields.Str()
    last_name = fields.Str()
    date_of_birth = fields.Date()
    orders = fields.List(fields.Nested(OrderSchema))

    @post_load
    def make_person(self, data, **kwargs):
        return Person(**data)

# Now you can use the schemas to load/dump data
person_schema = PersonSchema()

# Let's assume we have some JSON data
json_data = {
    'first_name': 'Jane',
    'last_name': 'Smith',
    'date_of_birth': '03/16/1977',
    'orders': [
        {
            'created': '2024-01-25T15:00:04',
            'amount': '100.00'
        }
    ]
}
# Deserialize the JSON data to Python dataclass
person = person_schema.load(json_data)
print(person)

The make_order and make_person methods decorated with @post_load are a part of marshmallow's deserialization process, which ensures the data is loaded into the dataclass after being deserialized.

Note: as_string=True is passed to fields.Decimal because if you were to load the value straight from JSON, you would get a float, which could cause precision issues. This way, the value is loaded as a string and converted to Decimal, keeping precision intact.

You can also use marshmallow_dataclass package to automate the process of creating schemas from dataclasses:

from decimal import Decimal
from dataclasses import dataclass
from datetime import date, datetime
from typing import List
from marshmallow_dataclass import class_schema

@dataclass
class Order:
    created: datetime
    amount: Decimal

@dataclass
class Person:
    first_name: str
    last_name: str
    date_of_birth: date
    orders: List[Order]

# Generate Marshmallow Schemas from dataclasses
PersonSchema = class_schema(Person)()
OrderSchema = class_schema(Order)()

The class_schema function is a factory that generates a schema from dataclass.
