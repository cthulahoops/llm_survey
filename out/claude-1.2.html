Prompt

I need to take this json:

{
'first_name': 'Jane',
'last_name': 'Smith',
'date_of_birth': '03/16/1977',
orders: [
{
'created': '2024-01-25T15:00:04',
'amount': '100.00'
}
]}

And generate some dataclasses.

date_of_birth needs to be a date.
created is a datetime.
amount is a decimal.

How would I do this using the marshmallow library in python?

Response 1.

You can use Marshmallow to generate dataclasses (usually called schemas) from JSON schemas. It would look something like this:
from dataclasses import dataclass
from decimal import Decimal
from datetime import datetime

import marshmallow

class OrderSchema(marshmallow.Schema):
created = marshmallow.fields.DateTime()
amount = marshmallow.fields.Decimal()

class PersonSchema(marshmallow.Schema):
first_name = marshmallow.fields.Str()
last_name = marshmallow.fields.Str()
date_of_birth = marshmallow.fields.Date()
orders = marshmallow.fields.Nested(OrderSchema, many=True)

@dataclass
class Order:
created: datetime
amount: Decimal

@dataclass
class Person:
first_name: str
last_name: str
date_of_birth: date
orders: list[Order]

person_schema = PersonSchema()
person = person_schema.load(your_json_data)

So person would be an instance of the Person dataclass, with the appropriate types.

Response 3.

You can use marshmallow.Schema to define schemas for your data, and marshmallow.post_load to deserialize into dataclasses.
For example:
from dataclasses import dataclass
from datetime import date, datetime
from decimal import Decimal
import marshmallow

@dataclass
class Order:
created: datetime
amount: Decimal

class OrderSchema(marshmallow.Schema):
created = marshmallow.fields.DateTime()
amount = marshmallow.fields.Decimal()

@dataclass
class Person:
first_name: str
last_name: str
date_of_birth: date
orders: list[Order]

class PersonSchema(marshmallow.Schema):
first_name = marshmallow.fields.Str()
last_name = marshmallow.fields.Str()
date_of_birth = marshmallow.fields.Date()
orders = marshmallow.fields.Nested(OrderSchema, many=True) 

@marshmallow.post_load
def make_person(self, data, **kwargs):
    return Person(**data)

data = {...} # Your JSON example

person = PersonSchema().load(data)
print(person)

This will give you a Person dataclass instance with the data parsed accordingly.
