Prompt

I need to take this json:

{
'first_name': 'Jane',
'last_name': 'Smith',
'date_of_birth': '03/16/1977',
orders: [
{
'created': '2024-01-25T15:00:04',
'amount': '100.00'
}
]}

And generate some dataclasses.

date_of_birth needs to be a date.
created is a datetime.
amount is a decimal.

How would I do this using the marshmallow library in python?

Response 1.

To generate dataclasses using the marshmallow library in Python, you can follow these steps:

  1. Install the marshmallow and marshmallow-dataclass libraries:
pip install marshmallow marshmallow-dataclass
  1. Define your dataclasses and schema using marshmallow-dataclass:
from dataclasses import dataclass
from datetime import date, datetime
from decimal import Decimal
from marshmallow import fields
from marshmallow_dataclass import dataclass as marshmallow_dataclass

@marshmallow_dataclass
class Order:
    created: datetime = fields.DateTime(data_key='created')
    amount: Decimal = fields.Decimal(data_key='amount')

@marshmallow_dataclass
class Person:
    first_name: str = fields.Str(data_key='first_name')
    last_name: str = fields.Str(data_key='last_name')
    date_of_birth: date = fields.Date(data_key='date_of_birth')
    orders: list[Order] = fields.List(fields.Nested(Order.Schema))

In this code:

  • We define two dataclasses: Order and Person.
  • For each field in the dataclasses, we specify the corresponding marshmallow field type and any necessary options (e.g., data_key to map JSON keys to dataclass field names).
  • The Order dataclass has fields for created (a datetime) and amount (a Decimal).

  • The Person dataclass has fields for first_name, last_name, date_of_birth (a date), and orders (a list of Order objects).

  • Use the schema to deserialize the JSON data into dataclass instances:

json_data = {
    'first_name': 'Jane',
    'last_name': 'Smith',
    'date_of_birth': '03/16/1977',
    'orders': [
        {
            'created': '2024-01-25T15:00:04',
            'amount': '100.00'
        }
    ]
}

person = Person.Schema().load(json_data)
print(person)

This will output:

Person(first_name='Jane', last_name='Smith', date_of_birth=datetime.date(1977, 3, 16), orders=[Order(created=datetime.datetime(2024, 1, 25, 15, 0, 4), amount=Decimal('100.00'))])

The marshmallow-dataclass library handles the mapping between JSON data and dataclass instances, automatically converting the values to the appropriate types (e.g., date, datetime, Decimal).

from decimal import Decimal
from typing import List

from marshmallow import fields, post_load
from marshmallow_dataclass import dataclass as marshmallow_dataclass

# Define nested dataclass for orders
@dataclass
class Order:
    created: datetime
    amount: Decimal

# Define schema for Order dataclass
class OrderSchema(marshmallow_dataclass):
    created = fields.DateTime(data_key="created")
    amount = fields.Decimal(as_number=True, data_key="amount")

# Define main dataclass
@dataclass
class Person:
    first_name: str
    last_name: str
    date_of_birth: date
    orders: List[Order]

# Define schema for Person dataclass
class PersonSchema(marshmallow_dataclass):
    first_name = fields.Str(data_key="first_name")
    last_name = fields.Str(data_key="last_name")
    date_of_birth = fields.Date(data_key="date_of_birth")
    orders = fields.Nested(OrderSchema, many=True, data_key="orders")

    @post_load
    def make_person(self, data, **kwargs):
        return Person(**data)

# Example JSON data
json_data = {
    "first_name": "Jane",
    "last_name": "Smith",
    "date_of_birth": "03/16/1977",
    "orders": [
        {
            "created": "2024-01-25T15:00:04",
            "amount": "100.00"
        }
    ]
}

# Load JSON data into dataclass
schema = PersonSchema()
person = schema.load(json_data)

print(person)

We then define two schemas: OrderSchema and PersonSchema. The OrderSchema maps the created and amount fields from the JSON data to the corresponding fields in the Order dataclass. The PersonSchema maps the first_name, last_name, date_of_birth, and orders fields from the JSON data to the corresponding fields in the Person dataclass. We use the fields.Nested field to handle the nested orders list.

Finally, we load the JSON data into the Person dataclass instance using the PersonSchema.
