'created': '2024-01-25T15:00:04',
'amount': '100.00'
}
]}

And generate some dataclasses.

date_of_birth needs to be a date.
created is a datetime.
amount is a decimal.

How would I do this using the marshmallow library in python?

Response 1.

You can use the marshmallow library in Python to create dataclasses that map the fields in the JSON data to corresponding marshmallow field class types. Here's an example:

from datetime import datetime
from decimal import Decimal
from marshmallow import fields, validate

class Person(fields.Model):
    first_name = fields.Str()
    last_name = fields.Str()
    date_of_birth = fields.DateTime(error_message="Invalid date of birth format")
    orders = fields.DateTime(error_message="Invalid orders field")

# Example usage
person_data = {
    'first_name': 'Jane',
    'last_name': 'Smith',
    'date_of_birth': '03/16/1977',
    'orders': '2024-01-25T15:00:04'
}

person = Person(**person_data)

# Validation will automatically check for errors
person.errors
# Output: orders: Invalid orders field
# In this case, the validate() decorator is used to validate the dates
person.date_of_birth.validate()
# Output: 03/16/1977
person.orders.validate()
# Output: 2024-01-25T15:00:04

In this example, we have defined a Person dataclass that maps the fields in the JSON data to corresponding marshmallow field class types. The DateTime field is used for the date_of_birth and orders fields. The validate() decorator is used to validate the date format. If the validation fails for a specific field, an InvalidField exception will be raised with an error message.

You can then create instances of the dataclass from the JSON data using the ** operator. The marshmallow validation is performed automatically during the serialization process, and errors are stored in the errors attribute of the Person instance. You can use the dataclasses module and the datetime and decimal modules to create dataclasses for the different data types in the JSON. Here's an example of how you can do this:

from dataclasses import dataclass, field
from datetime import datetime
from decimal import Decimal
import marshmallow

class Order(dataclass):
    @field(type=datetime)
    def created:
        pass

    @field(type=Decimal)
    def amount:
        pass

class Birthday(dataclass):
    @field(type=datetime)
    def date_of_birth:
        pass

@dataclass
class Person:
    first_name: str
    last_name: str
    date_of_birth: Birthday
    orders: List[Order]

class PersonSchema(marshmallow.Schema):
    first_name = marshmallow.Str()
    last_name = marshmallow.Str()
    date_of_birth = marshmallow.DateTime()
    orders = marshmallow.Schema(many=OrderSchema())

    class Meta:
        fields = ('first_name', 'last_name', 'date_of_birth', 'orders')

def from_json(person_json):
    person_schema = PersonSchema()
    data = person_schema.load(person_json)
    return person_schema.dump(data)

person_json = '{"first_name": "Jane", "last_name": "Smith", "date_of_birth": "03/16/1977", "orders": [{"created": "2024-01-25T15:00:04", "amount": "100.00"}]}'

print(person_obj)

  • Person: This dataclass has fields for the first_name, last_name, date_of_birth, and orders
