Prompt

I need to take this json:

{
'first_name': 'Jane',
'last_name': 'Smith',
'date_of_birth': '03/16/1977',
orders: [
{
'created': '2024-01-25T15:00:04',
'amount': '100.00'
}
]}

And generate some dataclasses.

date_of_birth needs to be a date.
created is a datetime.
amount is a decimal.

How would I do this using the marshmallow library in python?

Response 1.

To generate dataclasses using the marshmallow library in Python based on the provided JSON, you can follow these steps:

  1. Define the dataclasses:
from dataclasses import dataclass
from datetime import date, datetime
from decimal import Decimal

@dataclass
class Order:
    created: datetime
    amount: Decimal

@dataclass
class Person:
    first_name: str
    last_name: str
    date_of_birth: date
    orders: list[Order]
  1. Create marshmallow schemas for the dataclasses:
from marshmallow import Schema, fields, post_load

class OrderSchema(Schema):
    created = fields.DateTime(required=True)
    amount = fields.Decimal(required=True)

    @post_load
    def make_order(self, data, **kwargs):
        return Order(**data)

class PersonSchema(Schema):
    first_name = fields.Str(required=True)
    last_name = fields.Str(required=True)
    date_of_birth = fields.Date(required=True)
    orders = fields.List(fields.Nested(OrderSchema), required=True)

    @post_load
    def make_person(self, data, **kwargs):
        return Person(**data)

In the above code, we define two marshmallow schemas: OrderSchema and PersonSchema. These schemas correspond to the Order and Person dataclasses, respectively.

The OrderSchema has two fields: created of type DateTime and amount of type Decimal. The @post_load decorator is used to specify a method that will be called after deserialization to create an instance of the Order dataclass.

The PersonSchema has four fields: first_name, last_name, date_of_birth, and orders. The orders field is a list of nested OrderSchema instances. Similarly, the @post_load decorator is used to create an instance of the Person dataclass after deserialization.

  1. Use the schemas to deserialize the JSON:
json_data = {
    'first_name': 'Jane',
    'last_name': 'Smith',
    'date_of_birth': '03/16/1977',
    'orders': [
        {
            'created': '2024-01-25T15:00:04',
            'amount': '100.00'
        }
    ]
}

person_schema = PersonSchema()
person = person_schema.load(json_data)
print(person)

In this step, we create an instance of the PersonSchema and use its load method to deserialize the JSON data into a Person dataclass instance.

When you run the code, it will output:

The Person instance will have the first_name, last_name, date_of_birth, and orders attributes populated based on the JSON data. The date_of_birth will be a date object, the created field in the Order instance will be a datetime object, and the amount field will be a Decimal object.

This is how you can use the marshmallow library to generate dataclasses from the provided JSON, with the specified field types for date_of_birth, created, and amount.
