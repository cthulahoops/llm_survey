Prompt

I need to take this json:

{
'first_name': 'Jane',
'last_name': 'Smith',
'date_of_birth': '03/16/1977',
orders: [
{
'created': '2024-01-25T15:00:04',
'amount': '100.00'
}
]}

And generate some dataclasses.

date_of_birth needs to be a date.
created is a datetime.
amount is a decimal.

How would I do this using the marshmallow library in python?

Response 1.

You can create a set of dataclasses and then a Schema in Marshmallow to serialize and deserialize these classes. First, create a dataclass for each entry in the JSON:

from datetime import datetime
from dateutil.parser import parse
from decimal import Decimal
from typing import List, Dict
from dataclasses import dataclass

@dataclass
class Order:
    created: datetime
    amount: Decimal

@dataclass
class Person:
    first_name: str
    last_name: str
    date_of_birth: datetime
    orders: List[Order]

Next, define a Schema for each dataclass:

from marshmallow import Schema, fields, post_load
from dateutil.parser import parse
from decimal import Decimal

class OrderSchema(Schema):
    created = fields.DateTime(format='%Y-%m-%dT%H:%M:%S')
    amount = fields.Float(allow_nan=False)

class PersonSchema(Schema):
    first_name = fields.Str()
    last_name = fields.Str()
    date_of_birth = fields.DateTime(data_key='date_of_birth', serialize='strptime', deserialize='strftime', format='%m/%d/%Y')
    orders = fields.Nested('OrderSchema', many=True)

    @post_load
    def make_person(self, data, *args, **kwargs):
        return Person(**data)

Now you can use the PersonSchema for both loading and dumping. Make sure to parse your input date string before using it:

# Example input
json_input = {
  'first_name': 'Jane',
  'last_name': 'Smith',
  'date_of_birth': '03/16/1977',
  'orders': [
      {
          'created': '2024-01-25T15:00:04',
          'amount': '100.00'
      }
  ]
}

# Parse date of birth
date_of_birth = parse(json_input['date_of_birth'])

# Create Order objects and append them to an Order list
orders = [Order(datetime.strptime(order['created'], '%Y-%m-%dT%H:%M:%S'), Decimal(order['amount'])) for order in json_input.get('orders', [])]

# Create Person object and schema instance
person = Person(json_input['first_name'], json_input['last_name'], date_of_birth, orders)
schema = PersonSchema()

# Test loading from JSON
loaded_person = schema.load(json_input)
assert loaded_person.first_name == 'Jane'

# Test dumping to JSON
dumped_json = schema.dump(person)
assert dumped_json == json_input

This will allow you to handle the specific data types required in your json.

Response 2.

To achieve this, you can use Marshmallow's post_load hook to parse date and datetime strings upon deserialization and Marshmallow-Enum plugin to work with enums. First, you need to install Marshmallow-Enum plugin using pip install marshmallow-enum.

Here's an example of how to parse date_of_birth and amount, and use Marshmallow-Enum for OrderStatus:

from datetime import datetime
from decimal import Decimal
from marshmallow import Schema, post_load, Field
from marshmallow.validate import OneOf
from marshmallow_enum import EnumField


classOrderStatus = EnumField(Enum(OrderStatus))


class PersonSchema(Schema):
    first_name = Field(required=True)
    last_name = Field(required=True)
    date_of_birth = Field(required=True, allow_none=True, load_from='date_of_birth')

    class OrderSchema(Schema):
        created = Field(required=True, load_from='created')
        amount = Field(required=True, load_from='amount')
        status = classOrderStatus(required=True, allow_none=True)

    orders = fields.Nested(OrderSchema, many=True)


@post_load
def make_person(data, **kwargs):
    date_of_birth = data.get('date_of_birth')
    if date_of_birth:
        data['date_of_birth'] = datetime.strptime(date_of_birth, '%m/%d/%Y').date()
    return Person(**data)   


# Initializing the schema
person_schema = PersonSchema()

# Loading the data
data = {'first_name': 'Jane', 'last_name': 'Smith', 'date_of_birth': '03/16/1977', 'orders': [{'created': '2024-01-25T15:00:04', 'amount': '100.00'}]}
loaded_data = person_schema.load(data)

# Access the data
print(loaded_data.date_of_birth)
print(loaded_data.orders[0].created)
print(loaded_data.orders[0].status)

In this example, the make_person function will parse 'date_of_birth' into a datetime.date object and then create a Person instance with this data. The classOrderStatus serves to work with OrderStatus enum in Person.orders list.

You might have to adjust the code based on your actual data structure and requirements, but this should give you a general idea of how to use Marshmallow for this task.

Response 3.

You can use the Marshmallow library along with Python's built-in datetime and decimal modules to accomplish this. First, install the marshmallow library if you haven't already:

pip install marshmallow

Then, create a data class for the main person object:

from datetime import datetime
from decimal import Decimal
from marshmallow import Schema, fields, post_load

class Person:
    first_name = fields.Str(required=True)
    last_name = fields.Str(required=True)
    date_of_birth = fields.Date(required=True, error_messages={'missing': 'Date of birth is required'})

    orders = fields.Nested('Order', many=True, exclusive=True)

    class Meta:
        unknown = EXCLUDE

    @post_load
    def make_person(self, data, **kwargs):
        return Person(**data)

For the 'Order' data class, you'll need to convert the string timestamp to a datetime object:

class Order:
    created = fields.DateTime(alias='created', required=True, convert=lambda x: datetime.strptime(x, '%Y-%m-%dT%H:%M:%S%z'))
    amount = fields.Decimal(alias='amount', required=True, precision=2)

    class Meta:
        unknown = EXCLUDE necessarily unused

    @post_load
    def make_order(self, data, **kwargs):
        return Order(**data)
