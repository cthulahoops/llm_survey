Prompt

I need to take this json:

{
'first_name': 'Jane',
'last_name': 'Smith',
'date_of_birth': '03/16/1977',
orders: [
{
'created': '2024-01-25T15:00:04',
'amount': '100.00'
}
]}

And generate some dataclasses.

date_of_birth needs to be a date.
created is a datetime.
amount is a decimal.

How would I do this using the marshmallow library in python?

Response 1.

To achieve this using the marshmallow library, you can follow these steps:

  1. Install the marshmallow library if you haven't already:
pip install marshmallow
  1. Import the necessary modules from marshmallow:
from marshmallow import Schema, fields, post_load
from marshmallow.convertors import datetime, convert_datetime, decimal
from dateutil.parser import parse
  1. Define the dataclasses for the JSON structure:
class OrderSchema(Schema):
    created = fields.DateTime(load_only=True)
    amount = fields.Decimal(load_only=True)

class UserSchema(Schema):
    first_name = fields.Str()
    last_name = fields.Str()
    date_of_birth = fields.Date(load_only=True, data_key="date_of_birth_str", format="%m/%d/%Y")
    orders = fields.Nested(OrderSchema, many=True)

    @post_load
    def make_user_dict(self, data, **kwargs):
        user_dict = data.pop('user')
        user_dict['orders'] = data.pop('orders')
        return user_dict

class User(object):
    def __init__(self, **kwargs):
        self.user = UserSchema().load(kwargs)

    @property
    def first_name(self):
        return self.user['first_name']

    @property
    def last_name(self):
        return self.user['last_name']

    @property
    def date_of_birth(self):
        return parse(self.user['date_of_birth_str']).date()

    @property
    def orders(self):
        return self.user['orders']

    def to_dict(self):
        return self.user

json_data = {

        "last_name": "Smith",

            }
        ]
    }

user = User(**json_data)

Response 3.

To achieve this using the marshmallow library in Python, you can follow these steps:

  1. Install marshmallow:
pip install marshmallow
  1. Create a Python file called dataclasses.py and define your dataclasses as follows:
from dataclasses import dataclass
from datetime import datetime
from dateutil.parser import parse
from decimal import Decimal
import marshmallow

@dataclass
class Order:
    created: datetime
    amount: Decimal

@dataclass
class User:
    first_name: str
    last_name: str
    date_of_birth: datetime
    orders: list[Order]

class OrderSchema(marshmallow.Schema):
    class Meta:
        model = Order

class UserSchema(marshmallow.Schema):
    class Meta:
        model = User

    orders = marshmallow.Nested(OrderSchema, many=True)

    @marshmallow.post_load
    def post_load(self, data, **kwargs):
        date_of_birth_str = data.pop('date_of_birth')
        date_of_birth = parse(date_of_birth_str).date()
        orders = [Order(created=parse(order['created']), amount=Decimal(order['amount'])) for order in data['orders']]
        return User(**data, date_of_birth=date_of_birth, orders=orders)
  1. Use the UserSchema to load the JSON data into a User dataclass object:
json_data = {
  'first_name': 'Jane',
  'last_name': 'Smith',
  'date_of_birth': '03/16/1977',
  'orders': [
    {
      'created': '2024-01-25T15:00:04',
      'amount': '100.00'
    }
  ]
}

user_schema = UserSchema()
user = user_schema.load(json_data)

print(user)

This should output:

User(first_name='Jane', last_name='Smith', date_of_birth=datetime.date(1977, 3, 16), orders=[Order(created=datetime.datetime(2024, 1, 25, 15, 0, 4), amount=Decimal('100.00'))])

Note: The post_load method is necessary to convert the date_of_birth and created strings to datetime and date objects, respectively, and to convert the amount string to a Decimal object.
