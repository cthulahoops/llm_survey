Prompt

I need to take this json:

{
'first_name': 'Jane',
'last_name': 'Smith',
'date_of_birth': '03/16/1977',
orders: [
{
'created': '2024-01-25T15:00:04',
'amount': '100.00'
}
]}

And generate some dataclasses.

date_of_birth needs to be a date.
created is a datetime.
amount is a decimal.

How would I do this using the marshmallow library in python?

Response 1.

To use the marshmallow library to generate dataclasses with field data types, follow these steps:

  1. Define the dataclasses for the JSON structure.
  2. Implement schema classes for the dataclasses using marshmallow.
  3. Load the JSON data and dump it into the dataclasses.

Here's the code to achieve this:

from datetime import date, datetime
from decimal import Decimal

from dataclasses import dataclass
from marshmallow import fields, Schema, post_load

@dataclass
class Order:
    created: datetime
    amount: Decimal

@dataclass
class Person:
    first_name: str
    last_name: str
    date_of_birth: date
    orders: list[Order]

# Orders schema
class OrderSchema(Schema):
    created = fields.DateTime()
    amount = fields.Decimal()

# Person schema
class PersonSchema(Schema):
    first_name = fields.Str()
    last_name = fields.Str()
    date_of_birth = fields.Date()
    orders = fields.Nested(OrderSchema, many=True)

    @post_load
    def create_person(self, data, **kwargs):
        return Person(**data)

# Create an instance of the schema
person_schema = PersonSchema()

# JSON data to load
json_data = {
    'first_name': 'Jane',
    'last_name': 'Smith',
    'date_of_birth': '03/16/1977',
    'orders': [
        {
            'created': '2024-01-25T15:00:04',
            'amount': '100.00'
        }
    ]
}

# Load JSON data and dump into the dataclasses
person = person_schema.load(json_data)

print("First name:", person.first_name)
print("Orders:", person.orders)

This code defines the Person and Order dataclasses, as well as the corresponding PersonSchema and OrderSchema. It then creates an instance of PersonSchema, loads the JSON data, and converts it into a Person object. You can access the data field as person.first_name, person.date_of_birth, person.orders, etc.

Note that your JSON date_of_birth string needs to be in a suitable format, e.g., '1977-03-16' or '1977-03-16T00:00:00.000000Z', for marshmallow.fields.Date to properly parse it. In the example provided, you have '03/16/1977', which may not be parsed correctly.
